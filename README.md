# Git Practical Guide

This repository contains practical Git commands and workflows used in real-world development and DevOps environments.

The goal of this project is to demonstrate how to configure Git, authenticate securely using SSH keys, 
and perform common version control operations such as cloning repositories, pulling updates, and pushing changes.

## Topics Covered

- Git Installation and Configuration
- Generating and Using SSH Keys
- Connecting Git to GitHub using SSH
- Pushing Code to Remote Repository
- Cloning Repositories
- Pulling Latest Changes
- Basic Git Workflow
- Managing Branches
- Commit Best Practices

## Purpose

This repository serves as a quick reference for beginners and DevOps engineers who want to understand Git fundamentals and practice common commands used in daily development workflows.

## Technologies

- Git
- GitHub
- SSH Authentication
- Linux / Terminal
